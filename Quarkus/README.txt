To run the project just start the .bat file. It will install everything needed to run the project successfully. 
You can install manually if preferred.
- Latest Java
- Maven 
- Node.js 20
- Angular CLI

After that, enter on angular project folder (labseq-ui) in powershell and run "npm install" command to install node modules.
Then run "ng serve" command
Enter the code-with-quarkus folder and open powershell there and run "quarkus dev"

When everything is running you can open your browser and go to:
For API:
http://localhost:8080/q/swagger-ui/

For Frontend:
http://localhost:4200/
